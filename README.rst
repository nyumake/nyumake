 Hi there 👋
 ===========

*どのように* 
  用語は1行に収まるフレーズです。定義は、１つもしくは複数のパラグラフや
  その他の要素からなり、用語よりも深くインデントします。
  用語と定義との間に空行を入れてはいけません。

- 🔭 I’m currently working on ...
- 🌱 I’m currently learning ...
- 👯 I’m looking to collaborate on ...
- 🤔 I’m looking for help with ...
- 💬 Ask me about ...
- 📫 How to reach me: ...
- 😄 Pronouns: ...
- ⚡ Fun fact: ...
